# 我的苹果笔记本

## 零、序言

平时积累的东西以笔记的形式分享出来

## Usage

### 目录结构

```sh

- js
    - public 项目包
    - main.js 入口
    - package.json 依赖管理

- php
    - demo 各种栗子
    - ext php扩展包应用
    - container 容器
    - level 面试集合
    - bootstrap.php 入口
    - index.php 实例

```

### 使用

```sh
git clone https://github.com/OMGZui/noteBook.git

# 前端
cd js
npm install
node main.js
# 浏览器
http://localhost:5000/ 查看console


# PHP
composer install
php -S localhost:8000
# 浏览器
http://localhost:8000/php/index.php
# 命令行
cd php
php -f index.php
或
php index.php

```
